## 🛠️ `deployment_protocol`

### 🌐 Access the Matrix
Run the application directly in the cloud:  
🔗 **[Launch README Architect via AI Studio](https://ai.studio/apps/637cc25f-1247-4fbe-8f1b-e3c4bbc2f11f)**

### 🖥️ Run Locally
To deploy the application in your local environment, follow these steps:

**Prerequisite:** `Node.js` installed on your system.

```bash
# 1. Clone the repository and install dependencies
npm install

# 2. Configure environment variables
# Duplicate .env.example to .env.local and insert your API Key
echo "GEMINI_API_KEY=your_gemini_api_key_here" > .env.local

# 3. Ignite the server
npm run dev
```

### 📖 How to Use
1. **Input:** Paste your raw developer info, links, and projects into the Source Data terminal.
2. **Execute:** Click `Generate God-Tier README`.
3. **Deploy:** Preview the result, copy the raw Markdown, and push it to your GitHub profile repository.
